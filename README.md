# mod_hero
Hero - Joomla Module (SPEDI srl)

## Requisiti
 * Bootstrap 4

## Installazione
 * Per installare Grid Gallery scarica prima lo zip
 * Accedi al pannello di controllo di Joomla
 * Estensioni > Gestione > Installa
 * Installa il .zip
 * Ora troverai il modulo in gestione moduli
